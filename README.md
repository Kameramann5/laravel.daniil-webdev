**файлы для хостинга**
**Возможности новостного портала:**
**Админка:**
```
Категории: название, добавление, просмотр, редактирование, удаление
Теги: название, добавление, просмотр, редактирование, удаление
Статьи: название, описание, добавление, просмотр, редактирование, прикрепление изображения, можно указать теги из списка тегов, удаление
```
**Фронт:**

```
Просмотр списка статей, детальная статьи, вывод статей по категориям, вывод статей по тегам.
Сайдбар: вывести три статьи отсортировав по просмотрам, вывести список категорий и их количество.
Поиск статей.
Форма подписки.

Rest API
получить статьи, статью, вывести список новостей на главную, по кнопке показать полное описание поста.
ajax добавление статьa
ajax список статей и обновление при добавлении новой
ajax обновление статей
ajax удалить статью
```
